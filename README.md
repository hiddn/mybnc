## To compile
```
make
```

## Configuration

1. Edit **mybnc.allow**. It is _important_ that you restrict the ip addresses allowed to connect.
   * One per line;
   * Wildcards allowed;
   * CIDR not supported;
   * Hostnames not supported.

2. Edit **mybnc.conf** and follow the instructions written in comments.
